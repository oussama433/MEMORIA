# Version Angular du template Memoria

Ce dossier contient une adaptation Angular du template React présent dans `src/`.

## Ce qui a été converti

- Structure globale de layout (sidebar + zone contenu).
- Routage principal reprenant les mêmes chemins que `src/routes.ts`.
- Écran dashboard avec cartes KPI (exemple) + pages placeholders pour les autres routes.

## Lancer l'app Angular

```bash
cd angular
npm install
npm start
```

Puis ouvrir `http://localhost:4200`.
